Descargar e instalar Node.js:
* https://nodejs.org/es/download/
* Asegurarse que el "Instalar NPM Package Manager" esté tildado.

Entrar a la página de Sass y buscar el comando para instalar desde consola:
* 'npm install -g sass'
* Guía Sass: https://sass-lang.com/install

WIN 7
* Node 13.6.0 (https://nodejs.org/download/release/v13.6.0/)
* 'npm install -g sass'

MAC
* 'sudo npm'
* 'sudo npm install -g sass'

Instalar las extensiones de VSCode: 
- "Live Server" by Ritwick Dey
- "Live Sass Compiler" by Ritwick Dey
- (Opcional v1) "Easy Sass" by Wojciech Sura
- (Opcional) "Sass" by Syler
